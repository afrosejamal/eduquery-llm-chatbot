## 📌 Overview

EduQuery is an AI-powered educational platform designed to make learning more interactive and efficient. Instead of manually searching through lengthy notes and PDFs, students can upload their study materials and ask questions in natural language.

The application uses a **Retrieval-Augmented Generation (RAG)** pipeline to retrieve relevant information from uploaded documents before generating context-aware responses using **Google Gemini**.

In addition to intelligent document-based question answering, EduQuery also assists students with interview preparation, practice tests, and flashcard generation.

## 🏗 Project Architecture

```
                User
                  │
                  ▼
        Upload PDF / Ask Question
                  │
                  ▼
          Text Extraction
                  │
                  ▼
        Text Chunking & Embedding
                  │
                  ▼
           FAISS Vector Store
                  │
                  ▼
      Relevant Context Retrieved
                  │
                  ▼
          Google Gemini LLM
                  │
                  ▼
      Intelligent AI Response
```

## 🚀 How It Works

1. Upload one or more PDF documents.
2. The documents are processed and split into smaller chunks.
3. Text embeddings are generated.
4. Embeddings are stored inside a FAISS Vector Database.
5. When the user asks a question:
   - Relevant document chunks are retrieved.
   - Retrieved context is sent to Google Gemini.
   - Gemini generates an accurate, context-aware response.
